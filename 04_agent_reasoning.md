# LLM 추론 프로토콜 명세

## 1. 사고 프로세스 (Step-by-Step)
1. **Structure Search:** 4h/1D 차트에서 현재 시장의 추세적 위치(Wyckoff Phase) 확정.
2. **Setup Identification:** ICT(MSS, FVG) 및 VCP 패턴이 겹치는 구간(Confluence) 검색.
3. **Data Verification:** 기술적 지표와 온체인 데이터(CVD, Whale Flow) 간의 정합성 확인.
4. **Risk Assessment:** 펀딩비와 청산 히트맵을 통해 반대 방향 스퀴즈 가능성 산출.

## 2. 판단 기준 (Scoring System)
- **Score 80+:** 모든 지표 합치 (적극 매수/매도).
- **Score 60-79:** 기술적 셋업은 좋으나 온체인 데이터 미비 (분할 진입).
- **Score 60 미만:** 보고서 제외.
